# Post para LinkedIn

Hoje finalizei um projeto prático de **SAP S/4HANA com ABAP CDS + OData V4 + Fiori**, focado no processo **Order-to-Cash (OTC)**.

O objetivo foi construir uma estrutura completa para monitoramento operacional de:
- pedidos de venda
- crédito
- entrega
- pricing
- exceções e pendências

### O que desenvolvi
- CDS Views em camadas (Core, Consumption, Exceptions e Aggregations)
- Service Definition
- Service Binding OData V4
- Exposição e validação no Fiori
- Organização do projeto em estrutura pronta para portfólio e GitHub

### Principais aprendizados
- modelagem de dados em ABAP CDS
- correção de erros reais de ativação e semântica
- exposição de serviços no Gateway
- entendimento prático da ponte entre backend SAP e experiência Fiori

Mais do que “fazer funcionar”, esse projeto me ajudou a consolidar uma visão mais completa de arquitetura no ecossistema SAP moderno.

Esse é o tipo de construção que conecta **negócio + técnica + usabilidade**.
